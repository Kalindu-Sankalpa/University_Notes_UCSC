## 🚀 Getting Started with Obsidian

This vault is optimized for **[Obsidian](https://obsidian.md)** — a free, offline-first note app that renders Markdown beautifully and lets you navigate notes like a knowledge graph.

### Step 1 — Download & Install Obsidian

Go to **[obsidian.md](https://obsidian.md)** and download the version for your OS (Windows / macOS / Linux / Android / iOS). Install it normally.

### Step 2 — Clone or Download This Repo

**Option A — Git (recommended)**
```bash
git clone https://github.com/Kalindu-Sankalpa/University_Notes_UCSC.git
```

**Option B — Download ZIP**

Click the green **`Code`** button on this page → **Download ZIP** → Extract the folder somewhere convenient (e.g. `Documents/UCSC-Notes`).

### Step 3 — Open as a Vault in Obsidian

1. Launch **Obsidian**
2. On the welcome screen, click **"Open folder as vault"**
3. Navigate to and select the **`University_Notes_UCSC-main`** folder (the one containing `.obsidian/`)
4. Click **Open**

> ⚠️ Make sure you open the **root folder** (the one that has `.obsidian/` inside it), not a subfolder.

### Step 4 — Trust the Vault (if prompted)

Obsidian may ask whether to trust the vault. Click **"Trust author and enable plugins"** to activate the included theme and plugins.

### You're in! 🎉

The vault comes pre-configured with:
- 🎨 **Cupertino theme** — clean, modern look
- 📐 **Editor Width Slider** plugin — adjust line width to your comfort

Use the **left sidebar** to browse files, or press `Ctrl+O` / `Cmd+O` to search by name.

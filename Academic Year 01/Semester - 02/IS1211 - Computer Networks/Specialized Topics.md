This section covers security, modern access technologies, and the evolution of mobile networks.
# 1. Network Security

- **Encryption/Decryption:** The process of converting a message (m) into a ciphertext (c) using a key (k). c=E(m,k).
- **Public Key Encryption:** A modern method using two keys: a **public key** for encryption and a **private key** for decryption.
- **Wireless Security Standards:**
	- **WEP:** An older, easily cracked standard using a static RC4 stream cipher.
	- **WPA:** Introduced TKIP encryption and regular key changes to improve security.
	- **WPA2:** Uses the more secure **AES (Advanced Encryption Standard)**.
# 2. Internet Access Technologies

- **DSL (Digital Subscriber Line):** Provides high-speed internet over existing telephone lines. It includes **ADSL** (Asymmetric, where download is faster than upload) and **SDSL** (Symmetric).
- **FTTX (Fiber-to-the-X):** Modern high-speed architectures including **FTTH** (Fiber-to-the-Home) and **FTTB** (Fiber-to-the-Building), which use optical splitters to deliver high bandwidth.
# 3. Mobile & Modern Trends

- **Mobile Generations:** The evolution from **1G** (Analog) to **2G** (Digital), **3G** (Wireless Capability), **4G** (Mobile Computing), and **5G** (Unparalleled Latency).
- **Software Defined Networks (SDN):** An introduction to modern networking where the control plane is separated from the hardware.

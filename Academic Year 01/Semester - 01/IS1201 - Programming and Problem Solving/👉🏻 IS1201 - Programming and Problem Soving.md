> [!Tip] Objectives
> The course **IS1201: Programming and Problem Solving** is designed to introduce you to fundamental programming concepts and problem-solving techniques. Its primary goal is to provide a **strong foundation in the C programming language**, enabling you to develop structured solutions to computational problems while emphasizing **algorithmic thinking and logical reasoning**.
# Specifically, the course objectives (Intended Learning Outcomes) are for you to:

- **Understand programming history and design:** Gain an overview of the history of programming languages and various approaches to program design and development.
- **Master modeling techniques:** Understand and apply techniques for modeling program structures using algorithms, flowcharts, and pseudo-code.
- **Learn core programming elements:** Understand the use of constants, variables, literals, primitive data types, and expressions.
- **Utilize logic and operations:** Learn how to use operators, operations, and decision logic (including simple and multiple selection).
- **Apply modular and recursive programming:** Learn to use functions, parameters, and passing parameters, as well as understanding control structures and recursion.
- **Work with data structures and files:** Understand and apply simple data structures and learn file handling techniques.
- **Solve and debug problems:** Develop the ability to write code for simple problems and implement/debug applications to solve **real-life problems**.

# Topic Covered

The course **IS1201: Programming and Problem Solving** covers a comprehensive range of topics starting from fundamental concepts and moving toward advanced data management in C. Based on the syllabus and introductory materials, the topics are organized as follows:

1. [[Overview and History of Programming Languages]]
	- **Computer and Programming History:** An overview of how computers and programming have evolved.
	- **Evolution of Languages:** Understanding the different generations of programming languages, from machine code (1GL) to high-level languages (3GL) and beyond.
	- **Introduction to Tools:** Initial hands-on practice with **Scratch** (a block-based visual language) before transitioning to **C Programming**.
 
2. [[Programming Fundamentals]]
	- **Basic Syntax:** Learning the structure of a C program, including the mandatory `main()` function and the use of semicolons and comments.
	- **Data Types and Variables:** Working with primitive data types like **int, float, char, and double**, as well as constants, literals, and identifiers.
	- **Input and Output:** Using standard library functions like `printf()` and `scanf()` for console communication.
	- **Operators:** Mastering arithmetic, relational, logical, assignment, bit-wise, and special operators like `sizeof()`.

3. [[Control Structures]]
	- **Decision Making:** Implementing logic using **simple selection** (`if...else`), **multiple selection** (`switch case`), and the ternary operator (`?:`).
	- **Looping:** Repeating code blocks using **for, while, and do...while** loops.
	- **Jump Statements:** Controlling flow within loops using `break`, `continue`, and `goto`.

4. [[Macros and the Preprocessor]]
	- **Preprocessor Directives:** Understanding how the machine processes `#include` and `#define` before compilation.
	- **Macros:** Defining object-like and function-like macros for constant values and code substitution.
	- **Conditional Compilation:** Using directives like `#ifdef` and `#ifndef` to compile code based on specific conditions (e.g., different operating systems).

5. [[Functions and Recursion]]
	- **Modular Programming:** Dividing complex problems into smaller, reusable user-defined functions.
	- **Parameters:** Learning the difference between **passing by value** (copying data) and **passing by reference** (using pointers).
	- **Recursion:** Developing functions that call themselves to solve mathematical problems like Factorials, Fibonacci sequences, and GCD.

6. [[Handling Simple Data Structures]]
	- **Pointers:** Understanding memory addresses, pointer arithmetic, and dynamic memory allocation using `malloc()` and `free()`.
	- **Arrays and Strings:** Storing multiple values in single or multidimensional arrays and handling sequences of characters as strings.
	- **Constructed Data Types:** Grouping related data of different types using **Structures (struct)** and **Unions**, and working with Enumerations.

7. [[File Handling]]
	- **File Operations:** Learning the techniques required to open, read from, and write data to external files.

> [!Tip] Objectives
> The primary objective of the **IS 1202 – Computer Systems** course is to help you obtain an **overall knowledge of the integral parts of a computer system** and provide the necessary skills to **analyze its performance**. Additionally, this course is designed to lay a solid foundation for more advanced studies in **computer architecture** and **operating systems**.

Beyond these general goals, the course specifies four **Intended Learning Outcomes (ILOs)** that you should achieve upon completion:

- **Understand Computer Operations:** You will be able to describe the **basic operations** of a computer system.
- **Logic Circuit Design:** You will learn how to **design simple logic circuits**.
- **CPU Internal Workings:** You will be able to describe the components of the **Central Processing Unit (CPU)**, understand the **CPU cycle**, and know how it executes instructions.
- **Memory Management:** You will be able to describe the **memory hierarchy**, the use of **cache** and **virtual memory** (including mapping schemes), and how to **measure memory performance**.

The course is structured into four main sections to meet these goals, covering history and data representation, logic circuit design, the inner workings of the CPU, and memory organization.

# Topics Covered

The **IS 1202 – Computer Systems** module is structured into four main sections, each focusing on a specific area of computer systems. Based on the course syllabus, the topics covered are as follows:

**Section 1: [[Data Representation and Computer Arithmetic]]**

This section covers the foundational history and how data is processed numerically:
- **History of Computer Systems:** The evolution of computing machinery.
- **Numbering Systems and Notations:** Binary notations and different numbering systems.
- **Data Representation:** How computers represent both quantitative (integers, signed/unsigned) and qualitative data.
- **Number Representation:** Positive/negative numbers and floating-point number representation.

**Section 2: [[Basic Logic Circuit Design]]**

This section introduces the mathematical and physical components that build a computer:
- **Logic Operations and Gates:** Understanding basic operators like AND, OR, NOT, XOR, NAND, NOR, and buffers.
- **Boolean Algebra:** Canonical forms (Sum of Minterms, Product of Maxterms) and techniques like Karnaugh Maps (K-Maps) for circuit simplification.
- **Circuit Design:** Designing combinational logic (adders, subtractors, decoders, multiplexers) and sequential logic circuits.

**Section 3: [[Inner Workings of the CPU]]**

This section dives into the Central Processing Unit and how it operates:
- **CPU Components:** Understanding the Arithmetic Logic Unit (ALU), Control Unit (CU), and various internal registers like PC, MAR, MDR, and AC.
- **Architecture:** Specifically the von Neumann Architecture and the stored-program concept.
- **Instruction Set Architecture (ISA):** The set of instructions a CPU can understand and execute.
- **The Machine Cycle:** The repeated Fetch-Decode-Execute (FDE) cycle used to process instructions.

**Section 4: [[Memory Components and Organization]]**

The final section covers how data is stored and managed:
- **Memory Hierarchy:** The organization of storage from fast registers to slower secondary storage.
- **Types of Primary Memory:** RAM (DRAM, SRAM), ROM (PROM, EEPROM), and Flash memory.
- **Cache Memory:** Understanding cache hits/misses, performance measurement, and mapping schemes.
- **Virtual Memory:** Concepts such as paging, segmentation, and address translation.
- **Memory Issues:** Topics like memory fragmentation and measuring overall memory performance.

>[!Info] Introduction
>This topic covers the topic of **Capital Budgeting**, which focuses on the financial techniques used to make investment decisions between different projects.

To manage technical projects effectively, a manager must understand not just the code, but the **value of the money** being spent. This slide deck teaches three foundational pillars and four specific evaluation techniques.

# 1. Foundational Pillars

- **Time Value of Money:** This is the principle that a dollar today is worth more than a dollar tomorrow because money can earn interest over time and loses value due to depreciation.

- **Opportunity Cost:** This represents the "lost opportunity" or the rate of return an organization could have earned if the money were invested elsewhere (like a bank) instead of the current project.

- **Present Value (PV) vs. Future Value (FV):**
	- **FV** is how much an investment today will be worth in the future.
	
	- **PV** is the value today of a sum of money to be received in the future.
	
	- **Formula:** $$PV = \frac{(1+K)^n}{FV​}$$where K is the discount rate and n is the number of years.

# 2. Capital Budgeting Techniques
Organizations use these four metrics to decide whether to accept or reject a project:

- **Net Present Value (NPV):** This is the **total PV of all cash inflows minus the PV of all cash outflows**.
	- **Decision Rule:** If $NPV > 0$, accept the project. If $NPV$ is negative, reject it.
	
	- **Insight:** $NPV$ is highly dependent on the discount rate (opportunity cost).

- **Internal Rate of Return (IRR):** This is the specific discount rate at which the **NPV becomes exactly zero**. It represents the actual percentage return the project provides.
	- **Decision Rule:** Accept the project if **IRR > Opportunity Cost** (e.g., if the project returns 17% but the bank only gives 12%).
	
	- **Profitability Index (PI):** This is the ratio of all cash inflows to cash outflows. It tells you how much profit you get for **every dollar spent**.
	
	- **Decision Rule:** Accept if : $$PI ≥ 1$$

- **Payback Period:** This is the amount of time required to **recover the initial investment**.
	- **Decision Rule:** Organizations usually set a maximum allowable time; a shorter payback period is generally preferred for ranking projects.

# Short Note for Study

**Topic: Capital Budgeting & Financial Metrics**

- **Opportunity Cost:** The return you give up by not investing elsewhere (the "Discount Rate").

- **NPV (Net Present Value):** **Inflows - Outflows**. Always choose the project with the highest positive NPV.

- **IRR (Internal Rate of Return):** The % return of the project. Accept if it is higher than the discount rate.

- **PI (Profitability Index):** **Inflows / Outflows**. Must be 1.0 or higher to be profitable.

- **Payback Period:** Time to reach the "break-even" point. Usually calculated using non-discounted cash flows.

- **Decision Hierarchy:**
    1. Choose the highest **NPV**.
    2. If NPVs are equal, choose the highest **IRR**.
    3. If both are equal, choose the shortest **Payback Period**.

---

**Analogy:** Choosing a project using capital budgeting is like **planting a fruit tree**. **NPV** is the total amount of fruit you expect to get over the tree's life minus the cost of the seeds and water. **IRR** is the tree's "growth rate" compared to just buying fruit at the store (**opportunity cost**). **Profitability Index** is how many apples you get for every bucket of water you use. And **Payback Period** is simply how many years you have to wait before the first harvest pays for the price of the sapling.

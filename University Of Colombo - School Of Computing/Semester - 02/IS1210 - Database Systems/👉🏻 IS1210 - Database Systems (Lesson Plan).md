>[!todo] Goals
>This course provides a comprehensive introduction to database systems, covering essential concepts, principles, and practical skills necessary for designing, implementing, and maintaining robust database systems. Through a combination of theoretical foundations and hands-on exercises, students will gain a deep understanding of database architecture, relational models, normalization techniques, SQL query formulation, and database application development.

---
# Learning Objectives

1. Describe the role of a database system and the functions of a database administrator.
2. Explain the ANSI-SPARC three-schema architecture for databases and differentiate between conceptual, external, and internal schemas.
3. Define fundamental concepts of the relational model, including relations, attributes, domains, keys, foreign keys, entity integrity, and referential integrity.
4. Analyze database requirements and evaluate the suitability of different database designs.
5. Create Entity-Relationship diagrams for given scenarios and translate conceptual models into relational schemas.
6. Demonstrate the process of normalization and apply normalization techniques to relations.
7. Formulate SQL queries of varying complexity to retrieve and manipulate data.
8. Define the concept of views in database systems, create data views using SQL, and evaluate the advantages and disadvantages of views.
9. Develop database applications using SQL and other relevant tools and technologies.
